よく使われる技名まとめ

2019-05-19知識

マリオ64RTAでよく使われる技名をまとめました。(あまり使われない技名は後日まとめます)

プレイヤーが分からない単語を使っていた時にここを見ると良いでしょう。

汎用的なテクニック名は『基本テクニック』に載っているため割愛しています。各技のやり方などは『スター解説』でご確認ください。

※gif画像を多用しているためページをロードするのに時間がかかるかもしれません。

技名の横にある★は難易度です。個人的な評価なので参考程度で見てください。

  • ★☆☆☆☆……やり方がわかれば安定するレベル
  • ★★☆☆☆……少し練習すれば安定するレベル
  • ★★★☆☆……時間をかければ安定するレベル
  • ★★★★☆……上級者であれば安定するレベル
  • ★★★★★……安定させることが難しいレベル

ボムへいのせんじょう

新ボム ★★★☆☆ / Bomb Clip

ボム兵バグを利用してワンワンの檻をすり抜ける技です。

Z抜け ★★★★☆ / Z Clip

ボム兵バグを利用してワンワンの檻をZボタン(しゃがみ)ですり抜ける技です。

Bボタン(ボム投げ)で抜けるよりもこちらのほうが速いですが、その分難易度も高いです。

慣性羽 ★★☆☆☆ / Sky Jump

坂を滑って慣性をつけることで羽マリオで直接島に乗る技です。

中腹空島 ★★★★☆ / Island Hop (ihop)

大砲や帽子を使うことなく、お山の中腹から空島に乗る技です。『中腹』と省略されることが多いです。

主に70枚RTAで使われ、この技を使うと城内移動を減らすことができるためにタイムが速くなります。

中腹三段 ★★★★★ / Triple Jump Island Hop

『中腹空島』を三段ジャンプダイブで行なう技です。

バッタンキングのとりで

セットアップぶっこわ ★☆☆☆☆ / Setup Cannonless

崖掴まりを利用することでほぼ100%の確率でぶっこわを成功させる技です。

テクスチャセットアップ ★★☆☆☆ / Texture Setup

テクスチャを目安にすることで『セットアップぶっこわ』の最初の過程をスキップする技です。

ノンストップぶっこわ ★★★★★ / OG Cannonless

セットアップなどを使わずノンストップでぶっこわスターを回収する技です。

三段かご ★★★☆☆ / Owlless

三段壁キックでかごの中に入る技です。

反転かご ★★★★☆ / Sideflip Owlless

反転壁キックでかごの中に入る技です。

二段かご ★★★★☆ / Double Jump Owlless

二段壁キックでかごの中に入る技です。

かいぞくのいりえ

ウツボ抜け ★★★☆☆ / Ship Clip

ウツボを出さずに船内に侵入する技です。「ギラティナ」「ウツボット」と呼ばれることもあります。

さむいさむいマウンテン

速達 ★☆☆☆☆

子ペンギンを素早く親ペンギンのもとへ届ける技です。

ピングートリック ★★☆☆☆ / Pingu Trick

親ペンギンに話しかける直前にZボタンで子ペンギンを離すことで、子ペンギンを置くアクションをカットする技です。

フライゴン ★★☆☆☆

バク宙で壁を蹴ってスターを取りに行く技です。

テレサのホラーハウス

扉抜け ★★★☆☆ / Door Clip

テレサのテキスト出現にマリオが硬直することを利用して扉を抜ける技です。「ドア抜け」と呼ぶこともあります。

テキストの仕様の関係で日本語初期版限定の技となっております。

テキストスキップ ★★★★★ / Text Skip

2匹のちびテレサを同時に倒すことでテキスト2つのうちの1つをカットする技です。

(通常は2つのテキストが出現します)

やみにとけるどうくつ

箱三段 ★★★☆☆ / Box TJ

天井に判定が無いのを利用して三段ジャンプで上からすり抜ける技です。

ムシャーナ ★★★★★/ BLJ Strat

上下リフトからのBLJで煙迷路をスキップする技です。

滑り台 ★★★☆☆

灰色の足場の斜面を滑って二段壁キックで赤コインを回収する技です。

ファイアバブルランド

(ストップ)サイホーン ★★☆☆☆ / Side Hop

三段壁キックで無理やり檻に掴まる技です。

赤スイッチを起動しない場合で使う限定技になります。(本スターは羽で取るほうが速いです。)

ノンストップサイホーン ★★★☆☆ / Side Hop

ノンストップでサイホーンをやる場合を特別に『ノンストップサイホーン』と呼びます。

たきのぼり ★★☆☆☆ / Lava Boost

火山内部にある滝をケツ燃やしで無理やり登る技です。

はばたき ★★★☆☆ / Long Jump Lava Boost

幅跳びでたきのぼりする場合を特別に『はばたき』と呼びます。『幅跳びたきのぼり』の略です。

幅跳びを開始する位置は2種類あり、画像は手前から幅跳びしているため『手前はばたき』と呼び、逆に、奥から幅跳びするのを『奥はばたき』と呼びます。

また、ケツを燃やす位置は2パターンあり、画像は速いパターンになります。

あっちっちさばく

ボムピラ ★★★★☆ / Pillarless (pless)

ボム兵バグを利用してピラミッドの上の入口に侵入する技です。

パターンがいくつかありますが、どれも難易度が高めです。

タマピラ ★★★★☆ / Tama Pillarless (tamapless)

ボム兵バグをする際、小山の手前に来ることなくボムピラをするのが『タマピラ』です。

RTAで使うボムピラの中で最も速く、発案者であるTama氏の名前を取って『タマピラ』と呼んでいます。

カンダタ ★★★☆☆ / Topless

一段壁キックでイワンテの入口に崖掴まりする技です。

ウォーターランド

潜水艦前 ★★★☆☆ / Front Sub

潜水艦を前から登る技です。

ラプラス ★★★★☆

100枚赤コインにて、赤コイン2, 3枚目で三段ジャンプを使う技です。

みずびたシティー

金網キック ★★★☆☆ / TJWK

三段で金網を壁キックすることで紫スイッチをスキップする技です。

「フェニックスキス」「オケツマン」と呼ばれることもあります。

先壊し ★★★☆☆ / Box First

エレベータスターにて先に箱を壊す技です。

たかいたかいマウンテン

インスタクリップ ★★★★★ / Insta Clip

下の水に着水した後、その反動で上側の水に触れることで一瞬で上側へワープする技です。

省略して「インスタ」と呼ばれることが多いのですが、インスタグラムのことではないので注意してください。

狙うことは簡単ですが、決まるかどうかは実質運なので、上位プレイヤーでも安定させることができないと言われています。

丸太蹴り ★★☆☆☆ / Log Wall Kick

ステージ中部にある丸太を壁キックする技です。「丸太キック」と呼ぶこともあります。

しげった ★★★★☆ / Breezeless

下の足場から三段壁キック+ヒップでスターのある足場に乗る技です。

サルバグ ★☆☆☆☆

サルを掴んだまま移動し、スターの檻付近にジャンプヒップで落とすことでサルの移動を大幅にカットする技です。

ちびでかアイランド

壁抜け ★★☆☆☆ / Mountain Clip

でか島の山の壁を抜けることを見たまま『壁抜け』と呼んでいます。

ハナちゃんスターやてっぺん箱スターで使われます。

チックタックロック

右ルート ★★☆☆☆

最初の視点から見て右に三段ジャンプするルートを『右ルート』と呼んでいます。

(マリオ自身からすると左に進んでいるのですが……。)

Moving Reds (ムービングレッズ) ★★★★☆

赤コインスターを仕掛け高速状態で回収する技です。

ステージに入場するパートを含めると、仕掛け高速状態で回収したほうが速いです。

Usamune Moving Reds (うさむねムービングレッズ) ★★★★☆

『Moving Reds』の進化バージョンです。

人間がRTAでできる限界の『Moving Reds』と言われています。

レインボークルーズ

ジュゲムルート ★★★☆☆ / Lakitu Bounce

ジュゲムを踏み台にして虹の船へ向かうルートです。

やみのせかいのクッパ

シゲルサイクル ★★★☆☆ / Shigeru Cycle

やみのせかいのクッパ 赤コインスターのRTAルートにおける旧最速周期です。

ステージ中部にある卵焼きの足場が固定周期で動いており、どの周期(cycle)に乗れるかでタイムが大きく変わります。

しあサイクル ★★★★☆ / Xiah Cycle

シゲルサイクルより0.4周期程度速いものを『しあサイクル』と呼びます。

上位プレイヤーであればほぼ100％乗れるぐらいの周期となっています。

月島サイクル ★★★★★ / Tsukishima Cycle

シゲルサイクルより1周期速いものを『月島サイクル』と呼びます。

人間がBLJ無しでできる最速周期となっています。

かなり難易度が高いため、極めているプレイヤーでもノーリセットで100%決めることは難しいと言われています。

ほのおのうみのクッパ

(ストップ)ポールバグ ★★☆☆☆ / Pole Glitch

ポールの下からポールの上の金網まで一瞬でワープする技です。

金網を利用したバグ技となっているため「金網バグ」と呼ぶこともあります。

ノンストップポールバグ ★★★★☆ / Non-Stop Pole Glitch

ノンストップでポールバグをやる場合を特別に『ノンストップポールバグ』と呼びます。

みみっちー ★☆☆☆☆

上下リフトの端っこでジャンプを出し灰色の壁を蹴る技です。

みみっちーそのものはとても簡単ですが、ポールバグ無しルートの最速周期でみみっちーを決める場合は難易度★★★★☆まで跳ね上がります。

※上下リフトが固定周期で動いており、一番速い周期に乗ることを最速周期と呼んでいます。

Ultimate FS (アルティメットルート) ★★★★★

ノンストップポールバグを決め最速周期に乗った上で、さらに速く進むルートです。

主に短距離カテゴリで使われますが、上級者でも安定させることは難しいと言われています。

※上下リフトが固定周期で動いており、一番速い周期に乗ることを最速周期と呼んでいます。

最速アチチ ★★★☆☆ / Early Cycle

赤コイン5枚目後、正面に幅跳びしケツを燃やすことで上下リフトの最速周期に乗る技です。

てんくうのたたかい！

まなもサイクル ★★★★★ / Manamo Cycle

1つ目の回転リフトで1周期(cycle)速いリフトに乗ることを『まなもサイクル』と呼びます。

まなも氏が最初にこの周期(cycle)を成功させたことが由来となっています。

シゲルート ★★☆☆☆

ステージ中盤でポールを経由するルートです。

安全ルートとして扱われていますが、意外と難易度が高いです。

(ストップ)ドサイドン ★★☆☆☆ / Left Side TJWK

ステージ中盤にある木の棒のような足場で三段壁キックを使い、ショートカットする技です。

左端でショートカットする場合は『ドサイドン』と呼びます。

ノンストップドサイドン ★★★☆☆ / Left Side TJWK

ノンストップでドサイドンをやる場合を特別に『ノンストップドサイドン』と呼びます。

画像では垂直に三段ジャンプを出していますが、斜めに三段ジャンプする『斜めドサイドン』という技もあります。

(ストップ)サイドン ★★★☆☆ / Right Side TJWK

ステージ中盤にある木の棒のような足場で三段壁キックを使い、ショートカットする技です。

右端でショートカットする場合は『サイドン』と呼びます。

ノンストップサイドン ★★★★☆ / Right Side TJWK

ノンストップでサイドンをやる場合を特別に『ノンストップサイドン』と呼びます。

その他

ジュゲムカット ★★☆☆☆ / Lakitu Skip

お城前の橋で発生するジュゲムの説明テキストをカットする技です。

LBLJ ★★★☆☆

お城のロビーでBLJをする技です。ロビーBLJを省略して『LBLJ』と呼びます。

主にやみのせかいのクッパに入るために使われる技ですが、ロビーにあるステージならどこにでも入ることができます。

この技を使うことでステージ攻略順を変えることができるので、16枚RTAの場合は30～40秒程度、120枚RTAの場合は8秒程度速くなります。

DDDスキップ ★★★★☆ / DDD Skip

SBLJというバグ技を使って、30枚スター扉とウォーターランドをスキップする技です。

ウォーターランドは英名(Dire, Dire Docks)の頭文字を取って"DDD"と呼ばれているため、『DDDスキップ』という技名となっています。

マリオの速度を見極めたうえで決められた入力を決める必要があるため、この技を決めるには色々覚えないといけません。

BSF ★★★☆☆

青スイッチを起動しつつそのままスターを回収することでテキストをスキップする技です。

『Blue Switch Fantasy』の略になっています。

BSFの赤スイッチ版『RSF』もありますが、RSFは難易度がかなり低く"技"という感じがないため、ほとんどそう呼ばれません。
